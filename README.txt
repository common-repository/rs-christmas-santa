== Description ==

Celebrate the holiday season with our WordPress plugin that includes Christmas Santa Pop-Up, Christmas Music,  Christmas Countdown , and  Christmas Schedule to keep your site merry and bright.

This  plugin is easy-to-use. Rs Christmas Santa plugin can add  santa, countdown, music to your WordPress website. All the effects are togglable so you can choose the ones you want.

== Installation ==

The quickest method for installing the Christmas Plugin is:

1. Visit Plugins -> Add Plugin -> Search  File "rs christmas santa "
2. Click "Install Now"
3. Finally click "Activate Plugin"

This section describes how to install the plugin and get it working.

e.g.

1. Upload `rs-christmas-santa.zip` unzip folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Privacy Policy and Terms of Service == 

This plugin interacts with an external server to fetch festive music.  
 `https://aptest.therssoftware.com/audio/` is used to host the music files . 
When the music feature is enabled, the plugin will request audio files from the server. 
Only the audio file name that needs to be downloaded is sent to the server.
